Statistical Software


  • jive: a Python package for dimensionality reduction for multiple data matrices (implements AJIVE).
  • ajive: an R package implementing AJIVE.

Code to Reproduce Results in Papers


  • svm_geometry: code that creates the figures from the Geometric Insights into SVM paper.
  • law_net: code for the experiments/figures for the paper on the Supreme Court citation network.

Other software


I occasionally release tutorials or small packages which I (and hopefully others) find helpful.
  • what_the_cluster: a python package implementing algorithms to determine the "optimal" number of clusters (e.g. gap statistic).
  • suprious correlations: a shiny app to demonstrate suprious correlations with UNC mens/womens basketball data for a middle/highschool science fair (this was collaboration with a number of others).
  • python implementation of a number of optimization algorithms
  • word_embed_tutorial: a tutorial in Python for getting started with word embeddings using a corpus of Supreme Court opinions.
  • ds_tutorials: exercies for getting started with doing data science in R/Python. Build to help undergads compete in DataFest .
