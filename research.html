Current Projects


  • Dimensionality reduction for network valued random variables
    Developing PCA and PLS methods for repeated network observations with applications to statistical connectomics. Collaboration with NeruoData.

  • Large scale network and text analysis of the American legal system
    Studying the structure and evolution of the law through the lens of the legal citation network using data provided by the legal non-profit CourtListener. We are interested in comparing and contrasting network based community detection with text based clustering. We are using traditional statistical text clustering algorithms (e.g. gaussian mixture models, latent dirichlet allocation) as well as deep learning based algorithms to cluster documents in a large text corpus. See my group's github repo for more details.

  • Community weighted preferential attachment
    Stochastic processes and random network models.
